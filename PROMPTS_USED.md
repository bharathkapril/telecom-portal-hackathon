# Master Prompt Ledger
This document serves as the central repository for all Generative AI prompts utilized during the SDLC of the AI-Driven Telecom Plan Management Portal, ensuring full compliance with the Jury Evaluation Rubric.

### 1. Functional Requirements (FRD) & Epics
> **Prompt Used:** "Act as a Principal Software Engineer and Enterprise Solutions Architect. Generate Deliverable 1: Functional Requirements Document (FRD) for a Next.js/PostgreSQL Telecom Plan Management Portal. Outline the database schema (ER diagram logic) and the core user personas, epics, and user stories."

### 2. Architecture & Data Flow
> **Prompt Used:** "Act as a Cloud Solutions Architect. Generate Deliverable 2: Architecture Diagrams using Mermaid.js syntax. Create a Component Diagram showing the relationship between Next.js, the API, and PostgreSQL, and a Sequence Diagram mapping the exact data flow for a secure plan switch transaction."

### 3. Story-to-Spec Decomposition
> **Prompt Used:** "Act as a Technical Product Manager. Generate Deliverable 4: Story-to-Spec Decomposition (SSD). Decompose the requirements defined in the FRD into logical modules, epics, user stories, workflows, business rules, validation rules, and implementation-ready specification units."

### 4. Wireframes / UI Design
> **Prompt Used:** "Act as a UI/UX Designer. Generate Deliverable 5: Wireframes / User Interface Design for the Telecom Plan Management Portal. Structure this document to use screenshots of our actual Next.js application to demonstrate the Landing/Login View, Customer Dashboard, and Admin Dashboard."

### 5. SPECS / YAML Domain Specific Language
> **Prompt Used:** "Act as a Lead Systems Analyst. Generate Deliverable 6: SPECS / DSL in Markdown/YAML. Structure the specifications for each story component, including field definitions, screen behavior, validation rules, APIs, and a hierarchical file tree layout in strict YAML block format."

### 6. Code Generation (Next.js & Database)
> **Prompt Used:** "Act as a Senior Full-Stack Next.js Developer. Write the application code prioritizing security (Anti-IDOR), ACID database transactions, and Zod input validation. Please separate the monolithic UI into maintainable components inside a `components/` and `types/` folder architecture."

### 7. Code Quality AI Agent (Python)
> **Prompt Used:** "Act as an AI Tools Engineer. Build a Streamlit application in Python using the `google-generativeai` SDK. This agent must act as a strict Lead Developer performing offline Code Quality Reviews, specifically checking for Security vulnerabilities, Maintainability, and Error Handling."

### 8. Test Cases & Traceability
> **Prompt Used:** "Act as a QA Engineer & Project Manager. Generate Deliverable 8 (Functional Test Scripts) including Positive, Negative, Boundary, Exception, and Security tests, and Deliverable 9 (Traceability Matrix) mapping all epics to code and test cases."
